2.1 使用 OpenSlide 包获取图像的统计信息
  • OpenSlide
    是一个为阅读数字病理学中使用的高分辨率图片而创建的库。在处理这些特定的重图像时，它有很多实用的功能。这些图像在未压缩时可能占用数十 GB，因此无法使用标准工具或库轻松读取，这些工具或库专为可以轻松解压缩到 RAM 中的图像而设计。整张幻灯片图像通常是多分辨率的； OpenSlide 允许以最接近所需缩放级别的分辨率读取少量图像数据。
  • defaultdict()

    当字典中没有的键第一次出现时，default_factory自动为其返回一个空列表，list.append()会将值添加进新列表；再次遇到相同的键时，list.append()将其它值再添加进该列表。这种方法比使用dict.setdefault()更为便捷，dict.setdefault()也可以实现相同的功能。

原文链接：https://blog.csdn.net/chl183/article/details/107446836

2.2 Displaying images

显示图像并进行一些图像处理

2.2.1 Displaying random images using PIL

PIL代表Python Imaging Library，是最流行的用于图像打开和处理的python库之一（也称为pillow）。打开几个随机图像的时候，为了不占用 RAM，我们可以使用 PIL 函数 thumbnail() 快速查看。

请注意，PIL 将图像存储为 PIL 对象。这与将图像存储为 numpy 数组的 CV2 和 skimage 不同——这大大简化了后处理。



2.2.2 Displaying, resizing and manipulation using CV2 and skimage

使用 CV2 和 skimage 显示、调整大小和操作.这一部分将使用计算机视觉库 CV2 打开完整的图像。这个库也可以用于图像处理，我也会向您展示。您还可以使用 Canny 边缘检测器检测边缘。
注意：CV2 库有图像大小限制，并且会在我们数据集中的一些图像上出现错误。通过更改环境变量（参见第 0 节）有一种解决方法，但即便如此，某些图像也无法放入内存。这就是为什么我们稍后还要使用其他库的原因。
